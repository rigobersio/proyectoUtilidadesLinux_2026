Herramientas de Productividad para GNU/Linux (XFCE)

1. Historial de Portapapeles

Con CopyQ, puedes tener un historial completo de todo lo que copias. Esto te permite buscar y recuperar texto, enlaces o incluso imágenes que hayas copiado previamente, de una forma muy similar a la funcionalidad 'Windows + V'. 

# Comando de instalación (ej. Ubuntu)
sudo apt install copyq

# Comando para el atajo de teclado
copyq toggle

2. Captura de Texto (OCR)

Esta herramienta te permite "leer" el texto de cualquier parte de tu pantalla, aunque sea una imagen o un video. Simplemente seleccionas un área y el texto reconocido se copia automáticamente a tu portapapeles. 

# Comando de instalación (ej. Ubuntu)
sudo apt install maim tesseract-ocr xclip

# Comando del script ocr.sh
maim -s | tesseract stdin stdout | xclip -sel c

3. Selector de Color Global

Con gpick, puedes obtener el código de color exacto de cualquier píxel en tu pantalla. Es una herramienta indispensable para desarrolladores web y diseñadores que necesitan capturar colores de referencia rápidamente. 

# Comando de instalación (ej. Ubuntu)
sudo apt install gpick

# Comando para el atajo de teclado
gpick
